## 🚀 Overview

ThreatPilot AI is a modern cybersecurity platform that helps security professionals automate repetitive tasks and accelerate investigations.

Designed for:
- 👨‍💻 SOC Analysts
- 🔍 Threat Hunters
- 🛡️ Blue Teams
- 🎯 Penetration Testers
- 🎓 Cybersecurity Students

---

## ✨ Features

### ✅ IOC Extractor
Extract Indicators of Compromise from logs and reports.

Supported IOC Types:
- IP Addresses
- Domains
- URLs
- Email Addresses
- File Hashes
- CVEs

---

### 🤖 AI Log Analyzer *(Coming Soon)*

Analyze security logs using AI.

- Detect suspicious activity
- Explain alerts
- Recommend next steps
- Generate investigation summaries

---

### 📄 Incident Report Generator *(Coming Soon)*

Generate professional incident reports automatically.

- Executive Summary
- Timeline
- Root Cause
- Impact
- MITRE ATT&CK Mapping
- Recommendations

---

### 🔧 Cyber Utilities *(Planned)*

- Base64 Encoder/Decoder
- URL Decoder
- Hash Generator
- Password Generator
- Timestamp Converter
